## kuzco-llama installation
Spesifikasi Hardware


`Network Solana`
* 16GB RAM
* 30GB free disk space
* Reliable 10MB/s network bandwidth

### Register Site
```
https://kuzco.xyz/
```
### Install
```
curl -fsSL https://kuzco.xyz/install.sh | sh
```
### Runing
```
screen -S kuzco
```
```
kuzco init
```
### Useful Command

* Check Version
```
kuzco version
```
* Upgrade
```
kuzco upgrade
```
* Stopping a Worker
```
sudo kuzco worker stop
```
* Help
```
kuzco worker help
```
